# Open Trivia Visualizer

Visualization tool using data from the [Open Trivia DB API](https://opentdb.com).

Hosted at [https://kfpv.github.io/open-trivia-visualizer/](https://kfpv.github.io/open-trivia-visualizer/).

## Set up development environment

```bash
bun install
bun run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
